Karaokivy

How to use:

  • Install Kivy 1.8.0 '''master''' (or clone it, cythonize it and add it to your PYTHON_PATH)
  • Run "python main.py"

Note: It's not ready yet. At the moment it's able to play some .ogg audio files; it still needs improvement.
